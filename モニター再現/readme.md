# **AviUtl2 モニター再現スクリプトセット**

携帯ゲーム機の液晶モニター表示を再現するアニメーション効果スクリプト群です。

各ハードの液晶特性やピクセル配列をシミュレートします。

## **スクリプト一覧とパラメータ解説**

すべてのエフェクトは 画面効果\\モニター再現 内に追加されます。

### **1\. 解像度変更**

指定した携帯ゲーム機の解像度（アスペクト比）に合わせて仮想スクリーンを生成・リサイズします。

* **機種解像度**: 対象とするハードの基本解像度を選択します（GB/GBA/NDS/3DSなど）。「カスタム」を選択すると任意の解像度を指定できます。  
* **画面倍率**: 生成する仮想スクリーンの表示スケールを設定します。  
* **リサイズ方式**: 元映像のフィット方法を選択します（クリッピング / 内接 / 外接 / 比率無視）。  
* **中心X / 中心Y**: クロップやクリッピング時の中心位置を調整します。  
* **カスタム幅 / カスタム高さ**: 機種解像度で「カスタム」を選択した際のピクセル数を指定します。  
* **ドット補間(ニアレスト)**: チェックを入れると、ドット絵に適したニアレストネイバー補間でスナップします。

### **2\. GBモニター再現 (初代GB風.anm2)**

初代ゲームボーイのSTNモノクロ液晶の見た目を再現します。

* **ドットサイズ**: 液晶の1ドットあたりの大きさを設定します。  
* **グリッド濃度**: ピクセル間の隙間（ブラックマトリクス）の濃さを設定します。  
* **影の距離**: 液晶パネルと背景（反射板）との距離による影の深さを設定します。  
* **影の角度**: 影が落ちる方向を指定します。  
* **パレット**: 実機風（緑・グレー）、または3DSのバーチャルコンソール風のカラーパレットを選択できます。

### **3\. GBCモニター再現 (GBC風.anm2)**

ゲームボーイカラーの反射型カラー液晶の見た目を再現します。

* **ドットサイズ / グリッド濃度 / 影の距離 / 影の角度**: 上記GBモニター再現と同様です。  
* **環境光(明るさ)**: 外部からの光の強さをシミュレートし、画面の明るさを調整します。  
* **サブピクセル表示**: チェックを入れるとRGBサブピクセルの分離を描画します。  
* **15bitカラー化**: チェックを入れるとGBC特有の15bitカラー（各色5bit）に減色します。

### **4\. GBAモニター再現 (GBA風.anm2)**

ゲームボーイアドバンス歴代モデルの液晶の特性を再現します。

* **モデル**: 反射型の初代GBA、フロントライトのGBASP、バックライトのGBASPから液晶の特性を選択します。  
* **ドットサイズ**: 1ドットあたりの大きさを設定します。  
* **グリッド濃度**: ピクセルの隙間の濃さを設定します。  
* **サブピクセルとノッチ**: サブピクセルの描画と、GBA特有の画素形状をシミュレートします。  
* **15bitカラー化**: 実機同様の15bitカラーに減色します。

### **5\. NDSモニター再現 (NDS風.anm2)**

ニンテンドーDS世代のデュアルスクリーンの特性を再現します。

* **モデル**: 初代DS (NTR)、DS Lite (USG)、DSi LL (UTL) の液晶特性を選択します。  
* **画面**: 適用する画面（上画面 / 下画面）を選択します。下画面を選択した場合、タッチパネルの抵抗膜によるわずかな輝度低下や色温度の変化が加わります。  
* **ドットサイズ**: 1ドットあたりの大きさを設定します。  
* **ピクセル隙間濃度**: ピクセル間の網目の濃さを設定します。

### **6\. 3DSモニター再現 (3DS風.anm2)**

ニンテンドー3DSの液晶を再現します。裸眼立体視用の視差バリア構造もシミュレート可能です。

* **モデル**: TN液晶（標準）、IPS液晶、または2DS（スリット無）を選択します。  
* **画面**: 適用する画面（上画面 / 下画面）を選択します。  
* **ドットサイズ**: 1ドットあたりの大きさを設定します。  
* **3Dバリア濃度**: 上画面特有の3D視差バリア（黒いスリット線）の濃さを設定します。  
* **視差アングル(3Dシフト)**: 視点移動に伴うバリアのシフト量を調整します。  
* **サブピクセル表示**: サブピクセルの描画を有効にします。

### **7\. Game\&Watch風 (Game\&Watch風.anm2)**

ゲーム＆ウオッチなどのセグメントLCD（電卓液晶）を再現します。

* **2値化しきい値**: 元映像をモノクロに2値化する際のしきい値を設定します。  
* **影の距離**: 液晶セグメントから反射板へ落ちる影の深さを設定します。  
* **背景(反射板)**: 背景となる反射板の色を指定します。  
* **液晶(セグメント)**: 表示される液晶部分の色を指定します。
